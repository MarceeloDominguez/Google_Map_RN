## Run Locally

Clone the project

```bash
  git clone https://github.com/MarceeloDominguez/Google_Map_RN.git
```

Go to the project directory

```bash
  cd Google_Map_RN
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
